# Fast-Classified: Freelance Tutoring Marketplace

A full-stack web application connecting students with tutors for academic guidance and tutoring sessions.

## Tech Stack

- **Frontend**: React.js + Tailwind CSS + Vite
- **Backend**: FastAPI (Python)
- **Database**: PostgreSQL
- **Authentication**: JWT

## Features (Sprint 1)

- User authentication (Student/Teacher roles)
- Profile management
- Student help request feed
- Teacher discovery and search
- Real-time filtering

## Local Setup

### Prerequisites

- Python 3.9+
- Node.js 18+
- PostgreSQL 14+
